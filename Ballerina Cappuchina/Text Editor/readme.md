# Aplikasi Editor Teks Sederhana (Kelompok 6)

Aplikasi berbasis _Command Line Interface_ (CLI) dengan Python untuk mensimulasikan editor teks sederhana. Proyek ini merupakan tugas Proyek Akhir Mata Kuliah Algoritma & Struktur Data.

## Fitur Utama

- [cite_start]**CRUD Lengkap:** Menambah, membaca, mengubah, dan menghapus teks[cite: 13].
- [cite_start]**Undo Feature (Stack):** Membatalkan perubahan terakhir menggunakan prinsip LIFO (_Last In First Out_)[cite: 44, 48].
- [cite_start]**Penyimpanan Permanen:** Data teks disimpan secara otomatis ke dalam `dokumen.txt`.

## Cara Menjalankan

Buka terminal/console, lalu jalankan perintah:

```bash
python main.py

```
